# HPA-Achse (Hypothalamus-Hypophysen-Nebennierenrinden-Achse)

Zentrales neuroendokrines Stressregulationssystem. Verbindet Gehirn (Wahrnehmung) mit Körper (Hormonausschüttung).

## Anatomie / Schaltkreis

```
Stressor (real oder wahrgenommen)
        ↓
  HYPOTHALAMUS
  → CRH (Corticotropin-Releasing-Hormone)
        ↓
  HYPOPHYSE (Hirnanhangsdrüse)
  → ACTH (Adrenocorticotropes Hormon)
        ↓
  NEBENNIERENRINDE
  → CORTISOL
        ↓
  Zielorgane (Leber, Muskel, Immunsystem, Gehirn)
        ↓
  Negativer Feedback-Loop → Cortisol hemmt CRH + ACTH
```

## Funktion

- Mobilisiert Energie bei Stress (akut → adaptiv)
- Reguliert Immunantwort (anti-inflammatorisch kurzfristig)
- Beeinflusst Gedächtnis, Stimmung, Schlaf
- Tagesrhythmus: Morgenpeak (CAR), abends supprimiert

## Akut vs. Chronisch

| | Akuter Stress | Chronischer Stress |
|---|---|---|
| Cortisol | Kurz hoch → schnell normalisiert | Dauerhaft erhöht oder erschöpft |
| Immunsystem | Kurzfristig supprimiert | Chronisch dysreguliert |
| Feedback-Loop | Funktioniert | Gestört (Rezeptor-Downregulation) |
| Folge | Adaptiv, gesund | Burnout, Entzündung, Schlafstörung |

## Dysregulation (HPA-Dysregulation)

**Hyperaktiv** (Burnout Phase 1–2):
- Dauerhaft hoher Cortisol
- Schlafstörung, Bauchfett, Insulinresistenz, Angst

**Hypoaktiv** (Burnout Phase 3 / Erschöpfung):
- Paradox niedriger Cortisol (Rezeptoren erschöpft)
- Chronische Fatigue, flacher Morgenpeak, Immunsuppression

## Relevanz Ernährung (→ fuel-dev)

- Blutzuckerschwankungen aktivieren HPA → Cortisol steigt
- Koffein nüchtern potenziert morgendlichen CAR
- Magnesium ist Kofaktor für Cortisol-Synthese + -Abbau
- Chronisches Kaloriendefizit = metabolischer Stressor → HPA-Aktivierung
- Omega-3 dämpft CRH-Ausschüttung im Hypothalamus

## Relevanz Entspannung/PNI (→ relax-dev)

- HPA-Achse ist Kernthema der Psychoneuroimmunologie
- Chronischer Stress → HPA-Dysregulation → Immunsuppression → Entzündung
- Vagusnerv-Aktivierung (Parasympathikus) hemmt HPA indirekt
- Atemübungen, Meditation, Kälte senken CRH-Ausschüttung
- Schlaf ist wichtigster HPA-Reset: tiefer Schlaf → Cortisol-Talsohle
