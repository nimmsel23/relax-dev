# Kaffee / Koffein

Koffein (1,3,7-Trimethylxanthin) ist das meistkonsumierte psychoaktive Substanz weltweit.
Wirkung entsteht durch **Adenosin-Antagonismus** + sekundäre Kaskaden auf HPA, ANS und Neurotransmitter.

## Zusammensetzung (relevante Wirkstoffe)

| Substanz | Menge (1 Tasse Espresso ~30ml) | Wirkung |
|----------|-------------------------------|---------|
| Koffein | 60–80mg | Adenosin-Antagonist, Hauptwirkstoff |
| Chlorogensäuren | 50–150mg | Antioxidantien, verlangsamen Glukoseabsorption |
| Trigonellin | ~10mg | Neuroprotektiv, B3-Vorstufe |
| Diterpene (Cafestol, Kahweol) | nur ungefiltert | Cholesterin ↑ (Filterkaffee: minimal) |

## Primärmechanismus: Adenosin-Blockade

```
Adenosin (Müdigkeitssignal) → A1/A2A-Rezeptoren
Koffein blockiert diese Rezeptoren → kein Müdigkeitssignal
→ Dopamin + Noradrenalin wirken ungebremst
```

→ `neurotransmitters/adenosin.md`

## Kaskaden-Übersicht

```
KOFFEIN
│
├─ Adenosin-Antagonismus
│  ├─ → Dopamin-Aktivität ↑ (Motivation, Fokus)
│  └─ → Noradrenalin ↑ (Wachheit, Herzrate)
│
├─ HPA-Achse aktiviert
│  └─ → Cortisol ↑ (besonders nüchtern + Morgenpeak)
│
├─ Sympathikus-Aktivierung (ANS)
│  ├─ → Adrenalin aus Nebenniere
│  ├─ → Herzrate + Blutdruck ↑
│  └─ → Lipolyse (Fettmobilisierung)
│
├─ Melatonin-Suppression
│  └─ → Schlafonset verzögert (Halbwertszeit ~5–6h)
│
└─ Darm
   ├─ → Magensäure ↑
   ├─ → Motilität ↑ (Laxativeffekt)
   └─ → Mikrobiom: leicht präbiotisch (Chlorogensäuren)
```

## Timing & Halbwertszeit

- Wirkungseintritt: 15–45 min nach Konsum
- Peak-Plasmaspiegel: 30–60 min
- Halbwertszeit: **5–6 Stunden** (individuell 3–9h, CYP1A2-Polymorphismus)
- Kaffee um 14:00 → 50% noch um 20:00 aktiv

## Kontext-Abhängige Wirkung

| Kontext | Wirkung |
|---------|---------|
| Nüchtern + Morgen | Cortisol-Doppelpeak, intensiver Crash danach |
| Nach Mahlzeit | Cortisol-Antwort gedämpft, gleichmäßigere Wirkung |
| Chronischer Konsum | Adenosin-Rezeptor-Upregulation → Toleranz |
| Schlafentzug | Stärkere Wirkung, aber kein Ersatz für Schlaf |
| Zyklus (Frauen) | Lutealphase: langsamere Verstoffwechselung |

## Optimales Timing (evidenzbasiert)

- **Nicht sofort nach dem Aufwachen** — CAR (Cortisol Awakening Response) abwarten
- **Fenster: 90–120 min nach Aufwachen** — Cortisol sinkt, Koffein füllt Lücke
- **Cut-off: 10h vor Schlaf** (bei 6h HWZ und Schlafqualitäts-Sensitivität)

## Relevanz Ernährung (→ fuel-dev)

- Nüchtern-Kaffee + Cortisol-Peak = erhöhte Insulinresistenz am Morgen
- Chlorogensäuren verlangsamen postprandialen Blutzuckeranstieg (nach Mahlzeit)
- Koffein ist thermogen: Grundumsatz +3–11% kurzfristig
- Magnesium-Verlust durch Diurese (kompensieren)

## Relevanz Entspannung/PNI (→ relax-dev)

- Chronischer Koffeinkonsum = dauerhaft erhöhte Sympathikus-Aktivität
- Erhöht Angst-Bereitschaft (Adenosin hat anxiolytische Wirkung — blockiert = weniger)
- Melatonin-Suppression stört zirkadiane Rhythmik → HPA-Reset-Qualität sinkt
- Bei HPA-Dysregulation / Burnout: Koffein verschleiert Erschöpfungssignale

## Klienten-Erklärung

> "Koffein macht nicht wach — es blockiert das Müdigkeitssignal (Adenosin).
> Die Müdigkeit staut sich auf, kommt nach dem Crash zurück.
> Nüchtern morgens ist es wie Benzin ins Feuer: Cortisol ist schon hoch,
> Koffein pusht nochmal — und dann fällst du tiefer."
