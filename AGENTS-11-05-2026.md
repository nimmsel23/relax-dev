# 🤖 AGENTS.md

## 🧭 Projektkontext

Dieses Repository enthält ein Full-Stack TypeScript Projekt (React Frontend + Node.js Backend).

Es wird ein Feature implementiert:

> **Physio Timeline Graph Engine**

Ein Simulationssystem, das physiologische Reaktionen (Cortisol, Dopamin, Glucose) auf Lifestyle-Events (Coffee, Nicotine, THC, Meal) über Zeit modelliert und visualisiert.

---

# 🎯 Ziel des Agenten

Der Agent soll dieses Feature **vollständig funktional implementieren**, basierend auf `TODO.md`.

Dabei gilt:

* Fokus auf **Minimal Viable Product (MVP)**
* deterministische Simulation (keine ML/AI)
* klare Trennung von Backend, Engine und Frontend UI

# 🧠 Simulation Logik (Kernregel)

Für jeden Zeitschritt `t`:

```
finalCurve(t) =
  Σ eventEffect(t)
  × contextModifier
  + interactionEffects
```

### Events:

* coffee → cortisol ↑, dopamine mild ↑
* nicotine → dopamine spike
* thc → dopamine volatility ↑
* meal → glucose ↑

# 🔁 API Contract

## Request

```ts
{
  events: Event[]
  context: {
    fasted: boolean
    sleepDebt: number
    stressLevel: number
  }
  horizonMinutes: number
}
```

---

## Response

```ts
{
  timestamps: number[]
  curves: {
    cortisol: number[]
    dopamine: number[]
    glucose: number[]
  }
}
```
