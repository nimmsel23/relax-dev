# physio-dev

Physiologie Knowledge Base — gemeinsames Fundament für Ernährungs- und Entspannungstraining.

## Warum

Ernährung (fuel-dev) und Entspannung (relax-dev) sind zwei Interventionsebenen am gleichen System:
HPA-Achse, ANS, Hormonhaushalt, Neurotransmitter, Immunsystem.

Diese Knowledge Base erklärt das *Warum* hinter beiden Bereichen.

## Struktur

| Ordner | Inhalt |

| `knowledge/hormones/` | Cortisol, Insulin, Leptin, Melatonin, ... |
| `knowledge/neurotransmitters/` | Dopamin, Serotonin, GABA, Noradrenalin, ... |
| `knowledge/systems/` | HPA-Achse, ANS, Darm-Hirn-Achse, ... |
